계정과 연결된 서버 데이터를 삭제하고 더 이상 로그인할 수 없게 합니다.

삭제되는 내용

  • 소셜 로그인 연결, 모든 로그인 세션과 서버에 동기화된 목표·계획·기록·AI 크레딧 사용 내역은 요청 즉시 삭제됩니다.
  • 탈퇴 처리 중 재로그인을 막는 사용자 ID와 요청·삭제 예정 시각만 남긴 레코드에는 요청 시점부터 7일의 저장소 만료를 지정합니다.
  • 무료 체험 반복 발급 방지용 일방향 계정 식별값과 체험 사용 시각에는 체험 사용 시점부터 1년의 저장소 만료를 지정합니다.
  • 현재 브라우저에 저장된 목표·계획·기록도 요청 성공 직후 이 기기에서 삭제됩니다.
  • 다른 기기의 브라우저 저장 데이터는 서버에서 원격 삭제할 수 없으므로 해당 기기의 사이트 데이터에서 직접 삭제해야 합니다.

활성 정기결제가 있으면 결제사 해지를 먼저 확인하며, 실패하면 데이터 손실과 계속 청구를 막기 위해 탈퇴를 완료하지 않습니다. 계약·결제 기록이 있으면 이메일, 가명화 회원 참조값, 주문·결제 식별자, 구독 상태와 결제 시각을 일반 회원 레코드와 별도 키 영역에 논리 분리해 탈퇴 요청 후 5년 이내 보관할 수 있으며 빌링키는 보관하지 않습니다. 결제 기능 활성화 전에는 법정 보관 전용 저장소와 접근 권한을 별도로 구성할 예정입니다. 자세한 내용은 개인정보 처리방침을 확인해 주세요.

분산 저장소의 삭제 단계는 순서대로 실행됩니다. 모든 즉시 삭제 단계가 완료된 뒤에만 성공을 안내하며, 실패 메시지가 나오거나 이미 시작된 동기화·AI 요청과 경합한 경우 일부 단계 확인과 재시도가 필요할 수 있으므로 개인정보 문의로 알려 주세요. Apple 로그인을 활성화하는 경우에는 탈퇴 시 Apple token revoke 절차를 구현·검증한 뒤 제공합니다.
