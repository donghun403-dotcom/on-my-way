# On My Way PR Preview 실기기 QA 체크리스트

대상: Draft PR #3의 Cloudflare Preview URL. 테스트 계정만 사용하고 운영 계정·운영 결제수단은 사용하지 않는다. 각 항목의 결과는 `통과`, `실패`, `재현 불가`, `미검증` 중 하나로 기록한다.

## 공통 사전 조건

- Preview URL과 커밋 SHA가 PR의 최신 커밋과 일치하는지 확인한다.
- 브라우저의 기존 On My Way 쿠키와 사이트 데이터를 지운 상태, 기존 데이터가 있는 상태를 각각 한 번 수행한다.
- 계정 A와 계정 B는 Preview 전용 테스트 계정을 사용한다.
- Toss는 sandbox 키가 설정된 경우에만 수행한다. 실제 결제는 수행하지 않는다.
- 네트워크 로그에 세션 토큰, API 키, 결제 비밀키가 노출되지 않는지 확인한다.
